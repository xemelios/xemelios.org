Element TAdresse / TypAdr
Namespace No namespace
Annotations
 
Cette zone permet de déterminer l'adresse principale d'un tiers et les éventuelles adresses secondaires. Lorsque le type d'adresse est positionné dans le PES sur «non précisé», il sera entré dans Hélios en «principal» si la base ne contient aucune adresse principale ou sur «secondaire» si la base contient déjà une adresse principale.
