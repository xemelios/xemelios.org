Schéma des Etats de Frais

Ce schéma définit le format dans lequel les états de frais du SPL et de l'Etat (application CHORUS) doivent être transmis au comptable.

La dernière version du schéma est disponible ici.

Les spécifications associées sont disponibles au format PDF ou au format WORD.

Ce schéma est originellement basé sur le schéma de la facture UBL-invoice, aussi pour couvrir les besoins propre à l'état de frais, il se décompose en 2 parties :

  • Une partie qui permet d'englober plusieurs états de frais dans un seul élément (la documentation est disponible ici)
  • Une partie sous forme d'extension UBL-invoice, qui permet de regrouper tous les aspects métier de l'état de frais. (la documentation est disponible ici)
