Schéma des versements des bourses de l'éducation nationale

Ce schéma définit le format dans lequel les versements de bourses de l'éducation nationale doivent être transmis au comptable. Ce schéma est originellement basé sur le schéma de versement d'aide sociale, simplifié car les informations transmises sont plus simples.

La dernière version du schéma est disponible ici.
