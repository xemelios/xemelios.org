Schéma PES V2.1.a

Cette version du schéma PES est une évolution du PES V2, plus stricte et plus précise. Elle n'est pas encore finalisée.

  • elle ne définit que les parties réellement implémentées dans la chaîne de traitement,
  • les contrôles techniques sur les valeurs saisissables ont été renforcés pour simplifier la validation des flux,
  • elle prend en compte le format BIC/IBAN pour les comptes bancaires,
  • nous essayons, au fur et à mesure, de réintroduire la documentation dans le schéma, plutôt que de l'avoir dans un document annexé ; ce chantier n'est pas terminé.

La dernière version du schéma est téléchargeable ici.
